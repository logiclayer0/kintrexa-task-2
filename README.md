# Kintrexa Task 2

A premium, production-ready Blog Web Application built using the Flask framework and stylized with responsive Bootstrap dark theme components.

## Features
* **Dynamic Routing:** Multi-page web architecture using Flask routing.
* **Form Handling:** Secure client-side form submissions to dynamically append posts.
* **Elegant UI:** Responsive dark-mode dashboard with modern UI styling.
* **Vercel Deploy:** Ready-to-deploy architectural configuration.
